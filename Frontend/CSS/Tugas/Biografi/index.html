My Biografi

Saya lulusan Geofisika yang tertarik dengan pemrograman, sebelumnya saya pernah belajar bahasa pemrograman python. Saya juga pernah mengikuti program magang selama 3 bulan sebagai Fullstack Web Developer. Saat ini memilikiketertarikan di bidang web developer yang membuat saya mengambil langkah untuk belajar melalui beberapa Bootcamp, salah satunya di MERN Stack (MongoDB, ExpressJs, ReactJs, dan NodeJs). Di Bootcamp tersebut, saya belajar Backend dan Frontend dan memiliki beberapa proyek. Saat ini, saya membuka lowongan untuk posisi Fullstack Web Developer, Backend Developer, dan Frontend Developer, dan saya bersedia untuk pindah lokasi.
